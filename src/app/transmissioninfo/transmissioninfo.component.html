Boîte de vitesse

La boite manuelle est évidemment un inconvénient dans les bouchons, il est également assez de devoir changer l'embrayage pour les voitures kilométrées. Néanmoins elle présente certains avantages: elle permet de consommer moins en général, vous ne subissez pas les passages de vitesse qui peuvent procurer une sensation de désagréable selon le type de boîte automatique (à-coups, sensations de sentiments dans le vide) et vous maîtrisez les reprises de vitesse.

Les 2 roues motrices sont à l'arrière : elles permettent un meilleur angle de braquage (pratique en ville) et est assez prisé par les amateurs de conduite sportive.

Le contrôle de la voiture est également plus compliquée (l'électronique est là pour corriger vos trajectoires sur la plupart des modèles) voire impossible dans des conditions hivernales (neige, verglas).

Un 4x4 offre une meilleure adhérence mais aura tendance à consommer plus : toutes les roues sont motrices. La pose de pneus hiver est vivement recommandé voire obligatoire dans certains départements même si vous avez 4x4
